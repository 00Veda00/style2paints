説明

スケッチと参照イメージをアップロードするだけですが、参照イメージはスケッチ内の可能な限り詳細をカバーする必要があることに注意してください。 カラーピッカーを使用すると、参照画像から任意の色をクリックすることができます。 「pen」を使用して、尖ったヒントをスケッチに追加することができます。

嗜好

AIは、白い背景色を持つ参照画像を好むようです。 訓練データセットはアニメの絵画やガルガメCGに基づいているため、AIはこれらの分野でより優れた能力を持つ可能性があります。 正しい参照画像を選択することを忘れないでください。よりよい結果を得ることは常に役立ちます。
